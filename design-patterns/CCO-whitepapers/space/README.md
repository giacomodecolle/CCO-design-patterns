# Pattern Name
Facility Location

# Competency Questions

Where is a specific facility located?

What city is a facility located in, and what state is that city in?

What are all the facilities located within a particular city or state?

# Intent

To represent location relationships where facilities are situated within progressively larger geographic regions (e.g., building → city → state), enabling multi-level location queries and reasoning.

# Structure

Represents the location of a site and there approriate relations with a broader area.

Helps with creating transitive realtions between sites. 

Visual model through mermaid and png
